# CW_GameProgramming

School project for Computer Graphics Programming class.

We had to implement the following components:
- Particle Explosion
- Rigid Body Physics
- Crowd or flocking 
- A* Search
In order to have a full marks we had to do enhancement on components and interaction between them.
